Quick Start

Getting Started in 3 Minutes

  1. Open the Engineering Lab (the bolt.diy app)
  2. Pick a model from the dropdown (Gemini is free!)
  3. Type what you want to build — be specific! For example:
    "Build a physics simulation where I drop balls and they bounce off platforms. Add a slider to control gravity."
  4. Watch it build in real-time on the right side of the screen
  5. Keep iterating: "Add wind", "Make the balls different sizes", "Explain the physics code to me"
